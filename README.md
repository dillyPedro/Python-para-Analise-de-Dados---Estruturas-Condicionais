🚀 Projetos Iniciais em Python

Este repositório reúne projetos desenvolvidos durante meus estudos de Python, com foco em lógica de programação e estruturas condicionais.

Os projetos simulam situações reais e ajudam a consolidar conceitos fundamentais da linguagem.

📂 Projetos incluídos

🧮 1. Calculadora Inteligente

Aplicação que realiza operações matemáticas básicas:

Soma, subtração, multiplicação e divisão, e
validação de divisão por zero.
Classificação do resultado:
Par ou ímpar,
positivo, negativo ou zero e
inteiro ou decimal.

🔺 2. Verificador de Triângulos

Sistema que:

Recebe os lados de um triângulo, 
valida se é possível formar um triângulo,
classifica como:
Equilátero,
Isósceles e
Escaleno.

⛽ 3. Calculadora de Combustível

Simula um sistema de abastecimento:

Escolha entre etanol e diesel,
cálculo de desconto baseado na quantidade e
retorno do valor final a ser pago.

📊 4. Análise de Variação de Vendas

Ferramenta que:

Compara vendas entre dois anos,
calcula a variação percentual,
sugere ações estratégicas:
bonificação,
incentivo e
corte de gastos.

🧠 Conceitos aplicados:
Estruturas condicionais (if, elif, else),
operadores lógicos,
entrada e saída de dados,
validação de dados e
lógica de programação.

🛠️ Tecnologias utilizadas:
Python 3

🎯 Objetivo:

Praticar e consolidar os fundamentos da programação, desenvolvendo soluções para problemas do dia a dia e simulando cenários reais.

📌 Observação:

Este repositório faz parte do meu processo de aprendizado. Projetos mais avançados serão adicionados conforme minha evolução.
